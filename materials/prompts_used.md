# PROMPTS USED
## Introduction

During the project, AI tools were actively used through structured prompts designed for technical translation, terminology analysis, AI-assisted learning, and IT English practice.

The prompts below were refined and adapted according to modern prompt engineering principles:

- clear task definition;
- contextual grounding;
- role assignment;
- terminology focus;
- output specification;
- iterative refinement.

The prompts demonstrate practical AI-assisted workflows for technical language learning and professional translation support.

## 1. Technical Translation Prompt
### Purpose

Technical translation and adaptation of IT content into professional Russian technical language.

### Prompt

Act as a professional technical translator specializing in AI and IT terminology. Compare the English original and the Russian translation. 

#### Evaluate:

- technical accuracy;
- terminology usage;
- naturalness of Russian IT style;
- translation of technical collocations;
- semantic precision.

#### Identify:

- literal translation mistakes;
- terminology errors;
- unnatural Russian constructions;
- cybersecurity and cloud terminology issues.

#### Provide:

- detailed corrections;
- explanation of terminology;
- improved professional translation variants.

#### Special attention should be given to:

- hardware vs software distinction;
- enterprise terminology;
- cloud computing vocabulary;
- consumer technology terminology;
- cybersecurity concepts.

## 2. Glossary Extraction Prompt

### Purpose

Creation of a structured AI / IT glossary.

### Prompt

Analyze the provided technical text and extract all important AI / IT English terminology.

#### For each term provide:

- English term;
- Russian translation;
- short English explanation;
- short Russian explanation;
- contextual usage example.

#### Group terminology into categories:

- operating systems;
- cloud computing;
- cybersecurity;
- enterprise infrastructure;
- mobile technologies;
- networking;
- technical communication.

Include difficult technical collocations and terms frequently mistranslated by beginners.

## 3. Translation Error Analysis Prompt

### Purpose

Identification of common technical translation mistakes.

### Prompt

Act as an AI / IT translation reviewer.

#### Analyze the student’s translation and identify:

- terminology mistakes;
- literal translation problems;
- incorrect technical collocations;
- grammar problems affecting technical meaning;
- misunderstanding of enterprise or cloud terminology.

#### For every mistake provide:

- incorrect version;
- corrected version;
- explanation of the technical meaning;
- explanation of why the original translation is inaccurate.

#### Focus especially on:

- open-source terminology;
- cloud infrastructure;
- update loops;
- security vulnerabilities;
- seamless integration;
- enterprise systems;
- optimization terminology.

## 4. AI / IT Exercise Generation Prompt

### Purpose

Creation of technical English exercises.

### Prompt

#### Include:

- terminology translation exercises;
- error correction exercises;
- sentence translation tasks;
