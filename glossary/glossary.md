
# AI / IT Glossary

## Every Operating System Explained

This glossary contains key technical terms encountered during the project "Every Operating System Explained".

The glossary is intended to support:
- technical English learning
- AI / IT terminology acquisition
- translation practice
- technical communication skills.


## Operating systems
|                    Term                    |                     Russian translation                    | Description                                                                                |
|:------------------------------------------:|:----------------------------------------------------------:|--------------------------------------------------------------------------------------------|
|          User-friendly   interface         |             Удобный   пользовательский интерфейс           |            An interface that is easy to understand and use, even for beginners.            |
|       Graphical   User Interface (GUI)     |             Графический   интерфейс пользователя           |  An interface that allows users to interact with a system through windows, buttons, icons, |
|        Command-Line   Interface (CLI)      |                 Интерфейс   командной строки               |     A text-based interface that allows users to control a system by entering commands.     |
|     Hardware   and software integration    |     Интеграция   аппаратного и программного обеспечения    |             The optimized interaction between hardware and software components.            |
|                Compatibility               |                        Совместимость                       | The ability of a system to work with different hardware devices and software applications. |
|                 Flexibility                |                           Гибкость                         |   The ability of a system to be adapted and configured for different tasks and use cases.  |
|            Open-Source Software            |     Программное   обеспечение с открытым исходным кодом    |      Software whose source code is publicly available for inspection and modification.     |
|            Proprietary Software            |                      Проприетарное   ПО                    |               Commercial software with restricted access to its source code.               |
|            Distribution (Distro)           |                         Дистрибутив                        |   A version of Linux that includes a specific set of software, tools, and configurations.  |
|                Customization               |               Настраиваемость   / кастомизация             |             The ability to modify the appearance and functionality of a system.            |
